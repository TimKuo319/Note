
## JUnit5

+ `@BeforeAll`
	+ 需為static method
	+ 在執行所有Testcase前會執行一次
+ `@BeforeEach`
	+ 在跑每一個Testcase前都會執行一次
+ `@AfterEach`
	+ 在每一個Testcase執行完後執行一次
+ `@AfterAll`
	+ 需為static method
	+ 在所有Testcase執行完後執行一次
