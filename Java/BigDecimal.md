
在開發中如果遇到一些需要精確計算的部分時，因為電腦儲存結構的關係，使用 `double` 或`float` 就會造成計算上的失準，而為了解決這個狀況，Java 提供了 BigDecimal 這個方案，透過將浮點數轉換為字串儲存，來避免掉以上兩種結構儲存時的不精準。
