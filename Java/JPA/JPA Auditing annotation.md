

## JPA Auditing

JPA 提供了審計功能，可以自動追蹤實體的創建時間、修改時間、創建者和修改者等資訊。主要有兩套註解：

1. **Spring Data JPA 註解**：`@CreatedDate`、`@LastModifiedDate`、`@CreatedBy`、`@LastModifiedBy`
2. **Hibernate 註解**：`@CreationTimestamp`、`@UpdateTimestamp`
