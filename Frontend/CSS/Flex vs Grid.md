## What is `Flex and Grid` 

`Flex` 與 `Grid` 是常見的 CSS 佈局。兩者的差異在於 `Flex` 主要適用於單一維度的排版，比如單一 row、單一 column。

而 `Grid` 就如同他的名稱一樣，網格，主要適用於二維的排版。

因此在使用上，兩者通常會混用，利用 `grid` 來進行排版，針對頁面內的元件則透過 `flex` 來處理。

### justify-content-center 與 align-items-center 在 flex 中的應用

- `justify-content-center` 是針對主軸的置中
- `align-items-center` 是針對副軸的置中

而這裡的置中實際上是針對 layout 中的所有元素視為一個整體進行置中，去分配在主軸與副軸上的空間
