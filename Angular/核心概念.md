## Routing

依照不同的 url 路由，決定畫面上此時該顯示哪些 component


### 設定方式

#### 1. 先到 `app.routes.ts` 中進行路由設定

- `path` - 從根目錄開始路由的網址
- `pathMatch`
- `loadComponent` - 該路由需要載入的 component 

```ts
export const routes: Routes = [
    {
        path: '',
        pathMatch: 'full',
        loadComponent: () => import('./home/home.component').then(m => m.HomeComponent)
    },
    {
        path: 'todos',
        loadComponent: () => import('./todos/todos.component').then(m => m.TodosComponent)
    }
];
```

#### 2. 將 `<router-outlet/>` 放要使用的地方

網址在讀取時就會依照前面的 router 設定，在 `<router-outlet/>` 中依照 url 顯示對應的 component 

```ts
  template: `
    <app-header></app-header>
    <main>
      <router-outlet></router-outlet>
    </main>
      `,
```
