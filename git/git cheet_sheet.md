

+ 新增遠端資料庫
```sh
git remote add 
```

+ 設定遠端repository

```sh
git remote add <repo-name> <repo-URL>
```

+ 針對單一分支設定upstream

```sh
git branch -u <remote-upstream>
```

+ 合併不相關的兩個分支

```sh
git pull <remote repo> <branch name> --allow-unreleated-history
```

+ push到指定的分支

```sh
git push <repository alias> <branch name>
```
