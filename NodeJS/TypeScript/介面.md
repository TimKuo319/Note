TypeScript 中的介面本身相對於 Java 的介面來說更加靈活一些。

在 Java 中，介面通常用來抽象化某些行為來交由其他 class 去實作。而在 TypeScript 中，interface 可以拿來作為對**物件形狀的描述**。

```ts
interface Person {
    name: string;
    age: number;
}

let tom: Person = {
    name: 'Tom',
    age: 25
};
```
