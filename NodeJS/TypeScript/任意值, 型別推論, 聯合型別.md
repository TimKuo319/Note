## 任意值

在 typescript 中 `any` 用來表示任意型別，任意型別在被訪問任何屬性或方法時都不會報錯。

如果在宣告變數時未宣告型別，則會被視為任意值型別

```ts
let something;
something = 'seven';
something = 7;

something.setName('Tom');
```

## 型別推論

TypeScript 會在沒有明確的指定型別的時候推測出一個型別，這就是型別推論。

## Union Types

聯合型別（Union Types）表示取值可以為多種型別中的一種。

```ts
let myFavoriteNumber: string | number;
myFavoriteNumber = 'seven';
myFavoriteNumber = 7;
```

### 存取聯合型別的屬性或方法

當 TypeScript 不確定一個聯合型別的變數到底是哪個型別的時候，我們**只能存取此聯合型別的所有型別裡共有的屬性或方法**：

```ts
function getLength(something: string | number): number {
    return something.length;
}

// index.ts(2,22): error TS2339: Property 'length' does not exist on type 'string | number'.
//   Property 'length' does not exist on type 'number'.
```
