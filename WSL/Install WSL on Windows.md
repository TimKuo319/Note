
## 列出可安裝的wsl執行版本

```sh
wsl --list --online
```

## 選擇要安裝的wsl版本

```sh
wsl --install -d Ubuntu-22.04
```

之後會出現設定使用者帳號密碼的步驟，只要照著步驟填寫就可以完成WSL的安裝
